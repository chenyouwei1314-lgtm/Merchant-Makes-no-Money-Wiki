    奇梅亞 Chemeia

    δ 奇梅亞，實力最為弱小，通常群居生活，打倒後可得其素材 δ 核心。

    • 棘刺頭：以頭為主體，兩隻腳生於身體兩側，頭上佈滿棘刺。
    • 針刺頭：以頭為主體，兩隻腳生於身體兩側，頭上佈滿細長的針刺。
    • 噴嘴頭：外型如一顆水球，有個噴嘴用於發射體內的液體或氣體。
    • 軟鞭頭：以頭為主體，兩隻腳生於身體兩側，頭頂長出一根根細長的軟鞭。
    • 連枷頭：以頭為主體，兩隻腳生於身體兩側，頭頂長出一根根粗短的連枷。
    • 手球魔：身形矮胖，雙手的拳頭膨大如球。
    • 行走花：花朵的外型，利用淺根移動，為大型花的眷屬。
    • 注射者：肉塊的身形搭配細長的雙腿，有多個行動針頭用於注射不明液體。
    • 生產者：膨大的肉泥，會有各種 δ 奇梅亞不斷從其體內冒出。

    γ 奇梅亞，實力堅強，多為 δ 奇梅亞的老大，打倒後可得其素材 γ 核心。

    • 晨星魔：手球魔的進化體，拳頭膨大如球，並佈滿尖刺。
    • 軟鞭魔：軟鞭頭的進化體，為身形修長的人形怪物。
    • 連枷魔：連枷頭的進化體，為身形修長的人形怪物。
    • 章魚魔：頭部長出多根觸手，外型有如章魚一般。
    • 炮彈魔：單臂特化為砲管，當蓄滿能量後便能發射砲彈。
    • 重甲魔：全身佈滿硬殼，在行動上較為遲緩。
    • 巢穴花：擴展根部吸收植物養分，從而建立巢穴。
    • 棘刺獸：不僅全身，連頭部都佈滿棘刺。
    • 針刺獸：不僅全身，連頭部都佈滿細長的針刺。
    • 重甲獸：全身佈滿硬殼，唯獨腹部沒有。
    • 牢籠怪：牢籠的外型搭配多根觸手與腳，雖然本身行動緩慢，但是觸手攻速凌厲。
    • 長臂怪：雙臂異常地長，攻擊距離相當長。
    • 長爪怪：利用修長的爪子，撕開獵物深層的皮膚。
    • 大眼怪：一隻大眼能發射能量彈，以及能量射線。
    • 大嘴怪：一張大嘴和特化的牙齦，能輕易咬住、吞下一頭野豬。
    • 刺輪怪：輪子的身形搭配尖刺，靠著滾動折磨獵物。
    • 鬥劍者：雙臂特化為劍，有如戰士一般。
    • 尖叫者：發出尖叫聲，既能攻擊，還能呼朋引伴。

    β 奇梅亞，實力相當恐怖，通常單獨行動，打倒後可得其素材 β 核心。

    • 巨星魔：身形高胖，可以說是大型的晨星魔。
    • 多頭花：行走花的進化體，花朵錦簇，擁有各自的意志。
    • 寄生花：控制宿主的行動，同時吸收其養分。
    • 百足怪：蜈蚣的外型，全部的手一同攻擊會相當可怕。
    • 糜爛怪：勉強維持著人形，全身不停流著腐蝕性液體。
    • 融合體：將各種物品黏合，組成龐大的身體。
    • 火力狂：砲彈魔的進化體，擁有多根砲管。
    • 鋸肉者：鬥劍者的進化體，鋸齒狀的刀刃流露殘暴的本性。
    • 準王者：死亡的勇者成為奇梅亞的軀殼，再次拔起了劍。

    α 奇梅亞，為破壞神分裂的核心，依然有著毀天滅地的實力，打倒後可得其素材 α 核心。

    • 流星魔：巨星魔的老大，曾以招式「流星」將火山南部地帶夷為平地。
    • 女王花：吞噬無數的生命，其龐大的根部分支幾乎占據了一半的雨林。
    • 雷電獸：黑暗的雪山中閃爍藍色電光，那是死亡的訊號。
    • 腐朽體：有著多張嘴巴的腐爛球體，不斷製造沙塵暴。
    • 風暴龍：頑固的龍戰勝奇梅亞的支配，成為航海人的恐懼。
    • 前王者：墮落的戰王接觸禁忌，一發不可收拾。既是英雄，也是千古罪人。
