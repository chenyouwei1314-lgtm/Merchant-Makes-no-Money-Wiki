    奇梅亞 Chemeia

    δ 奇梅亞，實力最為弱小，通常群居生活，打倒後可得其素材 δ 核心。奇梅亞誕生於一場祕密的煉金術實驗，牠們被剝除了生物的機制，只保留侵略的本能。

    • 棘刺頭：以頭為主體，兩隻腳生於身體兩側，頭上佈滿棘刺。
    • 針刺頭：以頭為主體，兩隻腳生於身體兩側，頭上佈滿細長的針刺。
    • 噴嘴頭：外型如一顆水球，有個噴嘴用於發射體內的液體或氣體。
    • 軟鞭頭：以頭為主體，兩隻腳生於身體兩側，頭頂長出一根根細長的軟鞭。
    • 連枷頭：以頭為主體，兩隻腳生於身體兩側，頭頂長出一根根粗短的連枷。
    • 手球魔：身形矮胖，雙手的拳頭膨大如球。
    • 行走花：花朵的外型，利用淺根移動，為大型花的眷屬。
    • 注射者：肉塊的身形搭配細長的雙腿，有多個行動針頭用於注射不明液體。
    • 生產者：膨大的肉泥，會有各種 δ 奇梅亞不斷從其體內冒出。

    γ 奇梅亞，實力堅強，多為 δ 奇梅亞的老大，打倒後可得其素材 γ 核心。

    • 晨星魔：手球魔的進化體，拳頭膨大如球，並佈滿尖刺。
    • 軟鞭魔：軟鞭頭的進化體，為身形修長的人形怪物。
    • 連枷魔：連枷頭的進化體，為身形修長的人形怪物。
    • 牛頭魔：牛頭的外型，尖銳突出的大角用來貫穿獵物，脾氣相當火爆。
    • 章魚魔：頭部長出多根靈巧的觸手，有如章魚一般。
    • 砲彈魔：單臂特化為砲管，能發射砲彈，須蓄滿能量才能進行下一發。
    • 重甲魔：全身佈滿硬殼，只有關節處是裸露的。
    • 巢穴花：擴展根部以吸收周圍植物的養分，所在之處寸草不生。
    • 棘刺獸：不僅全身，連頭部和尾巴都佈滿棘刺。
    • 針刺獸：不僅全身，連頭部和尾巴都佈滿細長的針刺。
    • 重甲獸：全身佈滿硬殼，唯獨腹部和關節處沒有。
    • 長臂怪：雙臂異常地長，攻擊距離相當長。
    • 長爪怪：雙手特化出修長的尖爪，能撕開獵物深層的皮膚。
    • 大嘴怪：一張大嘴和特化的牙齦，能輕易咬住、吞下一頭野豬。
    • 牢籠怪：牢籠的外型搭配多根觸手與腳，雖然本身行動緩慢，但是觸手的攻速相當凌厲。
    • 砲台怪：身形巨大，能發射爆炸範圍極大的砲彈，不過行動非常緩慢。
    • 刺輪怪：輪子的身形搭配尖刺，靠著滾動身軀來折磨獵物。
    • 屍傀體：史萊姆的外型，專門操縱屍體。
    • 鬥劍者：下臂特化為劍，劍身斷裂時可迅速再生。
    • 尖叫者：發出足以傷害耳膜的尖叫聲，呼朋引伴。
    • 注視者：一隻大眼會持續注視著獵物，最終招致發狂。

    β 奇梅亞，實力相當恐怖，通常單獨行動，打倒後可得其素材 β 核心。

    • 巨星魔：身形高胖，可以說是大型的晨星魔。
    • 多頭花：行走花的進化體，花朵錦簇，擁有各自的意志。
    • 寄生花：控制宿主行動，並慢慢榨取養分，直到宿主死亡。
    • 百臂怪：有著多隻手臂，其招式「王八拳」惡名昭彰。
    • 糜爛怪：勉強維持著人形，全身流著腐蝕性液體。
    • 融合體：黏合各種物品，組成龐大的身體。
    • 火力狂：砲彈魔的進化體，手臂和背後特化出多根砲管，行動還是非常敏捷。
    • 鋸肉者：鬥劍者的進化體，鋸齒的刀刃流露殘暴的本性。
    • 準王者：死亡的勇者淪為奇梅亞的軀殼，再次拔起了劍。

    α 奇梅亞，為破壞神分裂的核心，依然有著毀天滅地的實力，打倒後可得其素材 α 核心。

    • 流星魔：巨星魔的老大，曾以招式「流星」將火山南部地帶夷為平地。
    • 女王花：吞噬無數的生命，其龐大的根部分支幾乎占據了一半的雨林。
    • 雷電獸：黑暗的雪山中閃爍藍色電光，那是死亡訊號。
    • 腐朽體：有著多張嘴巴的腐爛球體，不斷製造沙塵暴。
    • 風暴龍：頑固的龍戰勝奇梅亞的支配，成為航海人的恐懼。
    • 前王者：墮落的戰王接觸禁忌，一發不可收拾。
